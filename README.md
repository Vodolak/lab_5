
## Мета заняття
- Освоїти основи роботи з Git як розподіленою системою контролю версій.
- Навчитися створювати локальні та віддалені репозиторії.
- Зрозуміти принципи роботи staging area та фіксації комітів.
- Навчитися створювати та використовувати гілки для розробки нового функціоналу.
- Опрацювати методи злиття гілок та розв’язання конфліктів.
- Здобути навички співпраці через GitHub (push, pull, pull request).
- Навчитися працювати з SSH-ключами та персональними токенами доступу.
- Використовувати `.gitignore` для захисту конфіденційних даних.
